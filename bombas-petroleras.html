Bombas Petroleras

Sistemas BCP diseñados bajo normas API para la extracción, transferencia y tratamiento de hidrocarburos.

Nuestra línea petrolera está diseñada para soportar las condiciones más exigentes de los yacimientos (Upstream) y refinerías (Downstream). Alta eficiencia en el manejo de crudos pesados, con gas y abrasivos.

Línea NEA

Transferencia / Superficie

Bombas de Cavidad Progresiva horizontales para transferencia de crudo en baterías y oleoductos. Su diseño robusto permite el manejo de fluidos con alto contenido de sólidos y viscosidades variables.

Características:
  • Cumplimiento normas API 676
  • Caudales hasta 400 m³/h
  • Sellos mecánicos API Plan

Línea NEA Vertical

Inmersión / Semi-sumergible

Versión vertical de la línea NEA, ideal para instalación en piletas API, tanques de Slop o sumideros (Drain Sumps). Ahorra espacio físico y elimina problemas de NPSH disponible.

Características:
  • Longitud de columna variable
  • Placa base customizada
  • Mismo hidráulico que Línea NEA
